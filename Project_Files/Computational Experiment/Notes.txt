Stage 1 — Stimulus preparation and gold annotation. Formalize the event segmentation for each story. Create gold-standard event graphs (JSON format). Define the question
sets with expected answers. This is all pen-and-paper/spreadsheet work, no code needed yet, but it's the foundation for everything.

Stage 2 (analysis 3) — Prompted graph construction pipeline. Write the incremental prompting loop: for each segment, the model receives the current graph state plus the new segment
and outputs an updated graph. Implement JSON parsing and validation. Test with 2–3 stories and compare model-generated graphs against gold standard. Iterate on prompt design until
reliability is acceptable.

Stage 3 (analysis 2) — Hidden-state extraction and GNN graph prediction.
    - Forward pass each narrative through the model with output_hidden_states=True (no generation, just a plain forward pass).
    - Extract hidden states at layer x. For Qwen 2.5 7B: 28 transformer layers, hidden_dim=3584. Use a forward hook on a single layer if memory is tight.
    - Map token-level states to event-level node embeddings. Use span-based pooling over each segment's token range (mean/max/last-token).
      The existing segmentation boundaries give us natural event spans.
    - Train a GNN (e.g., GATv2Conv via PyTorch Geometric) on those node embeddings to predict edges (link prediction task).
      Edge decoder: concatenate [h_i, h_j, h_i * h_j] -> MLP -> predict edge class (no_edge / temporal / causal) + attribute heads for relation/distance/subtype.
      Gold-standard graphs provide supervision.
    - Sweep across multiple layers (e.g., 0, 7, 14, 21, 28) to profile when relational/temporal knowledge crystallises inside the model.
    - Sanity check: try a simple bilinear probe (score(i,j) = h_i^T W h_j) before the full GNN to see how much structure is linearly decodable.

Stage 4 (analysis 1) — Behavioural question-derived graphs. Feed the model the full narrative, then ask targeted behavioural questions (temporal ordering, causal attribution).
    Derive an event graph from the model's answers. This tests what the model can articulate explicitly, vs. what it represents internally (stage 3).

Stage 5 — Cross-level comparison. Compare graphs from all 3 levels of analysis (behavioural, hidden-state, prompted) against each other and against gold standard / human outputs.
    Key question: where do the levels diverge? Does the model internally represent structure it fails to articulate (gap between stage 3 and stages 2/4)?
